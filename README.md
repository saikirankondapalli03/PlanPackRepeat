# Installation

Prerequisites

* Java 8
* Maven

To Start

* Go to directory - `./code/travelapp-boot`
* Run ` mvn clean install`
* Go to `./code/travelapp-boot/target` 
* Run `java -jar travelapp-0.0.1-SNAPSHOT.jar` to start the application
